# springboot+vue校园二手书交易平台

#### 介绍
【技术栈】
①：架构: B/S、MVC
②：系统环境：Windows
③：开发环境：IDEA、JDK1.8、MySQL、Maven
④：技术栈：Java、MySQL、SpringBoot、Mybatis、MySql、VUE、jQuery


角色：管理员，用户

图书类型管理、公告类型管理、用户信息管理、个人信息管理、图书求购等
